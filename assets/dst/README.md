# 饥荒 (Don't Starve Together) 素材目录

## 动物素材

此目录存放饥荒游戏中的小动物素材，用于Obsidian宠物插件。

### 支持的动物类型

| 动物 | 文件名 | 描述 |
|------|--------|------|
| 🕷️ 蜘蛛 | `spider.png` | 普通蜘蛛 |
| 🦡 獾猫 | `catcoon.png` | 猫浣熊/獾猫 |
| 📦 切斯特 | `chester.png` | 切斯特箱子 |
| 🐷 猪人 | `pig.png` | 猪人/猪哥 |

### 图片来源

- [PNGkey - Dont Starve](https://www.pngkey.com/pngs/dont-starve/)
- [KindPNG - Dont Starve](https://www.kindpng.com/free/dont-starve/)
- [PNGwing - Dont Starve](https://www.pngwing.com/en/search?q=dont+Starve)

### 使用说明

1. 下载透明背景的PNG图片
2. 重命名为对应的文件名
3. 放入此目录
4. 重启Obsidian插件

### 素材规格建议

- 尺寸：128x128 像素
- 格式：PNG（透明背景）
- 风格：饥荒游戏原画风格
