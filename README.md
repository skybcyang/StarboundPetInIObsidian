# 🐱 Obsidian 宠物

在你的 Obsidian 笔记中养一只可爱的像素风宠物！

---

## ✨ 功能特性

- 🎨 **Starbound 像素素材** - 使用游戏级像素精灵图，精致可爱
- 🐱 **多种宠物类型** - 猫咪、兔子、萌兽、鼬等（逐步添加中）
- 🎨 **丰富外观选择** - 每种宠物有多种颜色/花纹变体
- 🚶 **精灵图动画** - Idle、Walk、Sleep 等多种动作动画
- 😴 **智能睡眠** - 长时间不操作后自动入睡，点击唤醒
- 💬 **对话气泡** - 点击右键打开聊天气泡，与宠物互动
- 📝 **一键转卡片** - 将对话记录保存为 Obsidian 笔记
- 💝 **互动反馈** - 点击抚摸，显示爱心动画
- 📁 **数据持久化** - 重启后宠物还在

---

## 🚀 快速开始

### 安装

1. 下载本插件的 `dist/` 文件夹内容
2. 复制到 Obsidian 仓库的 `.obsidian/plugins/obsidian-cat-pet/` 目录
3. 在 Obsidian 设置中启用插件
4. 使用命令面板（Ctrl/Cmd+P）输入「新建宠物」

---

## 🎮 使用方法

### 创建宠物

| 命令 | 效果 |
|------|------|
| `新建宠物` | 创建一只默认类型的宠物 |

### 在设置中选择宠物类型和外观

1. 打开设置 → 社区插件 → Obsidian 宠物
2. 选择「默认宠物类型」
3. 选择「默认宠物外观」
4. 新建的宠物将使用你选择的类型和外观

### 与宠物互动

| 操作 | 方式 |
|------|------|
| 抚摸 | 左键点击宠物 |
| 对话 | 右键 → 开始对话（打开聊天气泡） |
| 转卡片 | 聊天气泡中点击「📝 转卡片」按钮 |
| 改名 | 右键 → 改名 |
| 送走 | 右键 → 送走 |
| 显示/隐藏 | 命令面板 → 显示/隐藏宠物 |

### 宠物行为

- 🚶 **游走** - 宠物会在屏幕底部随机走动（带行走动画）
- 😴 **睡觉** - 长时间无操作后会入睡，点击唤醒
- 💝 **被摸** - 点击时会开心眨眼，显示爱心

## ⚙️ 设置选项

在设置面板中可以调整：

- **启用宠物** - 显示/隐藏所有宠物
- **最大宠物数量** - 限制同时显示的宠物数（1-10）
- **默认宠物类型** - 新建宠物时默认使用的类型
- **默认宠物外观** - 新建宠物时默认使用的外观
- **显示统计** - 在设置面板显示宠物列表
