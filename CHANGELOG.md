# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/).

## [4.0.0] - 2026-04-08

### ✨ Added
- Starbound pixel art sprite animation system
- 10 cat variants: tabby, black, white, calico, cream, grey
- Proper frame-by-frame animation for idle, walk, sleep, blink, jump
- Click to jump interaction with visual feedback
- Random jumping behavior based on pet personality
- Drag to move pets anywhere on screen
- Fixed all black label rendering issues

### 🔧 Fixed
- Stand label appearing in idle animation
- Sleep label rendering
- Jump animation black frames
- Pet movement speed synchronized with walk animation

## [3.2.0] - 2026-04-01

### ✨ Added
- Chat bubble interface for pet interaction
- Convert chat to note feature
- Pet renaming functionality

## [3.0.0] - 2026-03-15

### ✨ Added
- Initial simplified pet system
- Canvas-based cat rendering
- Basic idle and walk behaviors
- Click interaction with heart animation
- Data persistence across sessions
